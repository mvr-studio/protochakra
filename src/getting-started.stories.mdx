## Getting Started

First of all you have to install Chakra UI itself. Follow [this guide](https://chakra-ui.com/docs/getting-started).

Then install ProtoChakra using either NPM or Yarn.

```
yarn add @mvr-studio/protochakra
```

or

```
npm i @mvr-studio/protochakra
```
