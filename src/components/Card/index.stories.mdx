# Card

Card component provides an easy way to compose the card shadow without the need to override whole `boxShadow` property.

Properties that compose to `boxShadow`:

- shadowOffsetX
- shadowOffsetY
- blurRadius
- spreadRadius
- shadowColor
- shadowAlpha

However if you'd like to stick to `boxShadow` overriding as a whole property, then you can simply pass it as an argument or theme property.

```js
import { Card } from '@mvr-studio/protochakra'
```
